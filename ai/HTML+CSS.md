HTML и CSS **не являются языками программирования** в классическом понимании, так как они не содержат логики, переменных или алгоритмов вычислений. HTML — это [язык разметки](https://www.google.com/search?q=язык+разметки&ved=2ahUKEwjW_fjc6OiTAxWhIRAIHaUmKvsQgK4QegYIAQgAEAQ) (структура), а CSS — [язык стилей](https://www.google.com/search?q=язык+стилей&ved=2ahUKEwjW_fjc6OiTAxWhIRAIHaUmKvsQgK4QegYIAQgAEAU) (внешний вид), описывающий отображение документов. Они создают  статический каркас сайта, в то время как за интерактивность отвечает  JavaScript. 

**Основные отличия:**

- **HTML (HyperText Markup Language):** Размечает контент, создавая структуру: заголовки, параграфы, ссылки.
- **CSS (Cascading Style Sheets):** Стилизует элементы: цвета, шрифты, макеты. 

**Почему их не считают языками программирования:**

- **Отсутствие логики:** Нельзя выполнить вычисления или [циклы](https://www.google.com/search?q=циклы&ved=2ahUKEwjW_fjc6OiTAxWhIRAIHaUmKvsQgK4QegYIAQgBEAE).
- **Декларативность:** Они описывают, *как* что-то должно выглядеть или где находиться, а не *как* это обработать. 

Несмотря на это, в рамках веб-разработки их часто объединяют с JavaScript, образуя основу Frontend-разработки.
