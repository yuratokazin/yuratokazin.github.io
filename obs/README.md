# OBS

## OBS уже запущен но окна программы ни где нет  

Эта проблема обычно вызвана тем, что процесс OBS завис в фоне или свернулся в системный лоток (tray) KDE. Решение: принудительно завершить процесс  через терминал командой 

`pkill -9 obs` или проверить значок в трее. Если не помогает, переустановите OBS, предварительно удалив файлы конфигурации. 

**Основные способы решения в Debian/KDE:**

1. **Принудительное закрытие (самый верный способ):**
   Откройте терминал (Konsole) и введите:
   `pkill -9 obs`
   После этого попробуйте запустить OBS снова.
2. **Проверка системного лотка (Tray):**
   В KDE OBS часто сворачивается в трей (рядом с часами). Нажмите на  стрелочку «Показать скрытые значки» в панели задач и найдите иконку OBS, нажмите на нее правой кнопкой мыши -> «Показать» или «Развернуть».
3. **Удаление настроек (если OBS падает при запуске):**
   Иногда помогает удаление временных настроек, из-за которых окно не отображается:
   `rm -rf ~/.config/obs-studio`
4. **Перезапуск графической оболочки:**
   Нажмите `Alt+F2`, введите `kquitapp5 plasmashell && kstart5 plasmashell` (для Plasma 5) или аналогичную команду для Plasma 6, чтобы обновить панель задач.
5. **Переустановка (если ничего не помогло):**
   Удалите полностью:
   `sudo apt purge obs-studio`
   И установите заново.
