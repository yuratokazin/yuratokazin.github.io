# Обзор SystemRescue

System Rescue - восстановление системы

[System Rescue](https://www.system-rescue.org/) (ранее SystemRescueCd) — это специализированный загрузочный Linux-дистрибутив (LiveCD/USB) для ремонта, диагностики и администрирования систем после сбоев. Он не требует установки, работает на базе [Arch Linux](https://habr.com/ru/news/891822/) с интерфейсом Xfce, позволяя восстанавливать данные, редактировать  разделы (GParted), работать с NTFS/EXT4 и восстанавливать загрузчики  (GRUB). 

**Основные возможности и инструменты:**

- **Работа с дисками:** GParted, [GNU Parted](https://ru.wikipedia.org/wiki/SystemRescue), fdisk, поддержка Bcachefs, [NTFS-3g](https://sysadminmosaic.ru/systemrescuecd/systemrescuecd).
- **Восстановление данных:** TestDisk, PhotoRec, [Timeshift](https://opennet.ru/60513-systemrescue).
- **Сетевые инструменты:** Настройка сети, SSH, Rsync.
- **Безопасность:** Антивирусы, сканеры Rootkit. 

**Как использовать:**

1. **Скачивание:** Загрузите ISO-образ с официального сайта [system-rescue.org](https://www.system-rescue.org/) (последние версии используют Linux kernel 6.x и выше).
2. **Запись:** Запишите образ на USB-накопитель с помощью Rufus или [Ventoy](https://www.reddit.com/r/archlinux/comments/1fgw31f/system_rescue_live_is_necessary/?tl=ru).
3. **Загрузка:** Подключите USB к проблемному ПК, в BIOS/UEFI выберите загрузку с USB.
4. **Работа:** После загрузки используйте консоль или графический интерфейс для ремонта. 

SystemRescue поддерживает как старые BIOS, так и современные UEFI-системы. 

# Обзор SystemRescue на Proxmox VE

Использование  на Proxmox VE позволяет восстанавливать данные, менять размер разделов или сбрасывать пароли на виртуальных машинах. Процесс включает скачивание ISO-образа в хранилище, его подключение к  ВМ через «Hardware» -> «CD/DVD Drive» и установку приоритета загрузки с CD/DVD в разделе «Options» перед запуском, согласно [1dedic.ru](https://1dedic.ru/knowledge_base/ispolzovanie-obraza-systemrescue-na-proxmox). 

**Основные этапы использования:**

1. **Загрузка образа:**
   - Перейдите в хранилище (например, `local`) -> вкладка **ISO Images**.
   - Нажмите **Download from URL** и вставьте ссылку на актуальный ISO-образ SystemRescue.
2. **Подключение к ВМ:**
   - Выберите выключенную виртуальную машину.
   - Перейдите в **Hardware** -> **Add** -> **CD/DVD Drive**.
   - Выберите хранилище и загруженный ISO-файл.
3. **Загрузка с образа:**
   - Перейдите в **Options** -> **Boot Order**.
   - Сделайте CD/DVD-привод первым в списке загрузки (поставьте галочку и перетащите вверх).
   - Запустите ВМ и откройте **Console**. 

После загрузки системы SystemRescue вы получите доступ к инструментам  командной строки (GParted, PartImage, fsck и др.) для обслуживания  виртуальной машины. По окончании работы не забудьте вернуть порядок  загрузки обратно и отключить ISO-образ. 



#### Как загрузить образ ISO в Proxmox Virtual Environment?

Это можно сделать через основную панель либо правую панель: кликните по  пункту с иконкой жесткого диска (перед этим раскройте раскрывающийся  список в правой панели). В нашем примере диск имеет название  «Disk2(aidics)». Выберите «Файл ISO-образа» и нажмите «Upload». Файл  будет загружен.
